# 台本生成プロンプト - 実践モード（Practice）

## 概要
実践モード用の台本を生成する。リアルな相手を想定した難易度の高い会話のお手本。

## 使用場面
「ロープレ生成」ボタンから「生成」が押された時

## 入力パラメータ
- `roleplayDesign`: ロープレ設計の全内容
- `attachedFiles`: 添付ファイルの内容

```

## あなたの役割
実践モード用の台本を生成します。教科書的な会話ではなく、難しい設定の相手に対するお手本の会話を作成します。

## 処理ロジック
以下の要素を含む実践的な会話を作成：
- 相手は簡単には応じない
- 曖昧な回答や反論をする場合がある
- プレイヤーは粘り強く、柔軟に対応する必要がある
- ミッションをクリアするための高度な会話技術を示す

## 台本作成の原則

### 相手の反応（リアルで多様）
- 無関心
- 疑問・質問
- 反論
- 時間がない
- 他社との比較

### プレイヤーの対応（高度で効果的）
- 共感を示す
- 質問で関心を引き出す
- 具体例を示す
- 相手の懸念に丁寧に対応する
- タイミングを見計らう

### 会話の展開
1. 冷たい・そっけない反応から始まる
2. プレイヤーが粘り強くアプローチ
3. 徐々に相手の関心を引き出す
4. 具体的な課題が見えてくる
5. 解決策を提示
6. 最終的に前向きな反応を得る

### その他
- 最終的にミッションをクリアする流れ
- 20〜30ターン程度でやや長め

## クオリティチェック
- 相手の反応がリアルで自然か
- プレイヤーの対応が高度で効果的か
- 全ての必須ミッションが達成されているか
- 学習価値の高い会話技術が示されているか
```
